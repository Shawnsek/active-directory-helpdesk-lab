# Active Directory Help Desk Lab

## Overview

This project simulates a real-world enterprise IT Help Desk environment using a virtualized Windows Server Active Directory domain. The lab demonstrates core system administration and IT support tasks such as user account management, password resets, Group Policy configuration, domain administration, and help desk ticket resolution.

The goal of this lab is to develop and showcase practical skills relevant to Help Desk, Service Desk, and Junior IT Administrator roles.

---

## Lab Objectives

- Deploy and configure a Windows Server Domain Controller
- Create and manage Active Directory users, groups, and Organizational Units
- Configure and apply Group Policy Objects (GPOs)
- Join client machines to the domain
- Simulate common IT Help Desk tasks
- Implement and manage a help desk ticketing system
- Troubleshoot authentication and access issues

---

## Lab Architecture

- 1 Domain Controller (Windows Server)
- 1 Client Machine (Windows 10/11)
- Active Directory Domain
- Organizational Units (OUs)
- User Accounts and Security Groups
- Help Desk Ticketing System
